# 🛒 E-Commerce Website

A responsive full-stack e-commerce website developed using PHP, MySQL, HTML, CSS and Bootstrap.

## 🌐 Live Demo

https://kishenkc.infinityfreeapp.com

---

## Features

- User Registration & Login
- Customer Authentication
- Product Catalogue
- Shopping Cart
- Admin Dashboard
- Supplier Management
- MySQL Database Integration
- Responsive Layout
- Product Images
- Product Detail Pages
