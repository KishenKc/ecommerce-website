# 🛒 E-Commerce Website

A responsive full-stack e-commerce website developed using PHP, MySQL, HTML, CSS and Bootstrap.

## 🌐 Live Demo

https://kishenkc.infinityfreeapp.com

---

## Features

- User Registration & Login
- Customer Authentication
- Product Catalogue
- Shopping Cart
- Admin Dashboard
- Supplier Management
- MySQL Database Integration
- Responsive Layout
- Product Images
- Product Detail Pages

---

## Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- Bootstrap
- JavaScript

## Installation

1. Clone the repository

```bash
git clone https://github.com/KishenKc/ecommerce-website.git
```

2. Import the SQL database.

3. Configure the database connection.

4. Run the project using XAMPP or another PHP server.
